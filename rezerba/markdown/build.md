##<a id="sec3"></a>Markdownを使えるようにする  
Markdownを使えるようにするにあたりPackageを入れます  

[マークダウンエディタとして使う方法](http://futago-life.com/sublime-text3-wiki/how-to/markdown)  
上記のサイトから次のようなPackageをインストールしてください  

- OmniMarkupPreviewer  
- Monokai Extended  
- Markdown Extended  
- Trailing Space  
- Table Editor
